1. Configuración y Entorno:

He configurado la integración de IA generativa instalando de forma local Ollama y utilizando el modelo Llama 3 (específicamente llama3:8b).

He actualizado el archivo requirements.txt del equipo añadiendo la librería oficial de Ollama en Python para asegurar que el proyecto sea reproducible por el resto del equipo.

2. Clasificación Generativa y Prompt Engineering (generativo.py):

He desarrollado el script encargado de predecir el sentimiento de las opiniones (POSITIVO, NEGATIVO o NEUTRO) utilizando exclusivamente técnicas de Prompt Engineering.

Para encontrar la mejor estrategia, he diseñado y automatizado experimentos comparando tres enfoques distintos:

Zero-Shot: Instrucciones directas sin ejemplos.

One-Shot: Instrucciones aportando un único ejemplo de referencia.

Few-Shot: Instrucciones aportando un ejemplo de cada clase para guiar el formato de salida.

Ajuste técnico: He configurado la temperatura del modelo a 0.0 en estas pruebas para garantizar respuestas deterministas y evitar que la IA "alucine" o añada texto extra, forzándola a devolver estrictamente la etiqueta.

He programado la exportación de los resultados al entregable resultados_generativos.csv requerido, documentando el modelo, el prompt exacto, la entrada, la salida y el tiempo de ejecución, además de implementar la lógica para resaltar el mejor resultado.

3. Balanceo de Datos con IA / Oversampling Generativo (balancear_con_ia.py):

Para abordar el problema del desbalanceo de clases que afecta a los algoritmos tradicionales de mis compañeros, he creado un script de Oversampling Generativo.

He diseñado un prompt de paráfrasis con reglas estrictas para reescribir las reseñas de la clase minoritaria: debe mantener el mismo sentimiento exacto pero utilizar vocabulario y estructuras diferentes.

Ajuste técnico: A diferencia de la clasificación, aquí he subido la temperatura a 0.8 para darle "creatividad" al modelo y generar ejemplos variados y ricos, evitando duplicados exactos.

El script detecta automáticamente cuál es la clase minoritaria en nuestro train.csv, genera las opiniones sintéticas y crea el entregable con las paráfrasis generadas.
